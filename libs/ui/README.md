# ui

This library using for yoctoerp UI

## Install

- yarn add @yoctoerp/ui
- npm install @yoctoerp/ui
- import scss
  - import 'bootstrap/dist/css/bootstrap.min.css';
  - import 'bootstrap-icons/font/bootstrap-icons.css'
  - import '@yoctoerp/index.css'

## Require

- bootstrap
- bootstrap-icon
